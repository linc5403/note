# IDEA mac 快捷键

|    Shortkey         |   Action     |
|        -            |     -        |
Option + Enter          | 显示意向动作和快速修复代码
Command + /              | 注释 / 取消注释与行注释
Command + Option + /    | 注释 / 取消注释与块注释
Command + P              | 显示方法的参数信息
Control + J              | 快速查看文档
Control + H              | 显示当前类的层次结构
Command + O              | 跳转到Class
Command + Shift + O   | 跳转到文件
Ctrol + Alt + O       | 优化import
